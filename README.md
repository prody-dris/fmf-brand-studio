# FMF Brand Studio

An open naming exploration for the **Frappe Mobile Framework (FMF)** — an offline-first mobile framework on [Frappe](https://frappeframework.com) (Flutter SDK + Frappe server app).

Pick a candidate name from the dropdown and the whole page re-skins: palette, typography, design feel, and product philosophy all change to show the brand world each name would create. `FMF` stays as the constant descriptor; the chosen name sits on top (e.g. `mSaathi FMF`).

**Live page:** https://prody-dris.github.io/fmf-brand-studio/

## Candidates

| Name | Root | Idea |
|------|------|------|
| **mSaathi** | साथी — companion | The field worker's companion |
| **mSetu** | सेतु — bridge | Holds the connection across the gap |
| **mDisha** | दिशा — direction | Direction for every field |
| **Sahaj** | सहज — effortless | Field data, made effortless (the non-"m" option) |

Names are **under consideration, not final**. Availability (pub.dev, GitHub, trademark) needs verification before any name is adopted.
